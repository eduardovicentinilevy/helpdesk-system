# 🛠️ HelpDesk PHX Pro
**Sistema Full-Stack de Gestão de Suporte**

## 📝 Descrição
HelpDesk corporativo com autenticação JWT e níveis de acesso. Clientes abrem chamados com prioridade (Baixa a Alta) e técnicos gerenciam a fila global via dashboard com métricas em tempo real e histórico de auditoria. Design premium inspirado na paleta PHX Mahogany/Spruce.

## 🛠️ Tecnologias
* **Back-end:** Node.js, Express, JWT, MySQL.
* **Front-end:** Vanilla JavaScript, CSS3 (Custom Properties), FontAwesome.
* **Banco de Dados:** MySQL Workbench (Relacional com Auditoria).

## 🚀 Como Rodar
1. Importe o script SQL no MySQL.
2. Configure o `.env` com suas credenciais.
3. Instale as dependências: `npm install`.
4. Inicie o servidor: `node src/server.js`.
